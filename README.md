## 环境管理

本工程推荐使用`Python=3.9`，并建议另外新建conda环境，防止版本交叉污染：

```
conda create -n erniest python=3.9
```

Python的核心包版本：

```
streamlit                    1.35.0
erniebot                     0.5.8
streamlit-modal              0.1.2
```
